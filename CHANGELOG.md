# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-03
### Added
- Initial public release of the **MHD Xu-Auditor** pipeline.
- Core Xu-style modal convolution engine for spectral auditing.
- Support for `.npz` and `.npy` snapshots (vorticity/magnetic potential).
- Automated generation of paper-grade vector PDF figures.
- Metric suite: Blocking Index (BI), Tail Energy, and Regularity Ratio (rho).
- Algebraic consistency tests verifying $10^{-14}$ precision.
- CLI (Command Line Interface) for batch processing diagnostics.
